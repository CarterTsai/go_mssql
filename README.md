# go_mssql
golang mssql example

### Pre-install
    go get github.com/denisenkom/go-mssqldb
    
### Get Source 
    go get github.com/CarterTsai/go_mssql
  
### Install 
    go install github.com/CarterTsai/go_mssql
  
### Add Binary Path
    $> export PATH=$PATH:$GOPATH/bin

### Usage
    $> go_mssql -h
